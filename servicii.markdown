## Servicii de consultanță de mediu profesionale

Oferim o gamă completă de servicii de consultanță de mediu conform <a href="http://www.mmediu.ro/" target="_blank" rel="noopener noreferrer">legislației românești de mediu</a>. Fiecare proiect este adaptat nevoilor specifice ale clienților noștri.

### Expertiză în documentații de mediu

De peste 20 de ani, <a href="/echipa/">echipa noastră de experți</a> oferă soluții complete pentru reglementarea proiectelor pe linie de mediu. Beneficiem de certificări <a href="https://www.iso.org/iso-9001-quality-management.html" target="_blank" rel="noopener noreferrer">ISO 9001</a> și <a href="https://www.iso.org/iso-14001-environmental-management.html" target="_blank" rel="noopener noreferrer">ISO 14001</a>. Suntem înscriși în <a href="http://www.anpm.ro/" target="_blank" rel="noopener noreferrer">Registrul Național al Elaboratorilor de Studii pentru Protecția Mediului</a>.

Serviciile noastre acoperă întregul proces. De la evaluarea inițială conform <a href="https://ec.europa.eu/environment/eia/eia-legalcontext.htm" target="_blank" rel="noopener noreferrer">Directivei EIA</a> până la monitorizarea post-implementare. Lucrăm cu tehnologii de ultimă generație. Echipa noastră include biologi, ecologiști, <a href="/servicii/gis/">specialiști GIS</a> și <a href="/servicii/aerofotogrametrie/">operatori de drone certificați</a>.

### Portofoliu vast de proiecte finalizate

Am reglementat <a href="/proiecte/">peste 1000 de proiecte</a> la nivel național. Lucrăm în domenii variate: energie regenerabilă, infrastructură, industrie, dezvoltare urbană. Colaborăm cu instituții de cercetare și universități. Respectăm cele mai înalte standarde profesionale conform <a href="https://ec.europa.eu/environment/eia/sea-legalcontext.htm" target="_blank" rel="noopener noreferrer">Directivei SEA</a>.
